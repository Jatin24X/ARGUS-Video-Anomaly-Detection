ARGUS STREAM A - QUICK START

1. Open a terminal in this folder.

2. Install dependencies:
   pip install -r requirements.txt

3. Run the local demo:
   run_demo.bat

4. Primary Avenue benchmark report:
   outputs\reports\avenue_stream_a_best_test.json

5. Main technical references:
   README.md
   docs\methodology.md
   docs\evaluation.md
   docs\deployment.md
   technical_evaluation_summary.md
   demo_walkthrough.md
   technical_qa.md

6. Optional high-micro diagnostic report:
   outputs\reports\avenue_stream_a_best_test_high_micro_diag.json

Notes:
- The standalone package supports Avenue and UBnormal Stream A profiles.
- Avenue is the primary frame-level benchmark path.
- UBnormal is retained as a reference profile.
- The demo uses adaptive frame thinning and in-memory VideoMAE extraction for fast interactive analysis.
- Do not compare the full-frame Stream A result directly to object-centric Avenue scores without stating the protocol difference.
